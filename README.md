# 🌾 Agrinho 2026: O Uso da Tecnologia no Agronegócio (Agro 4.0)

Este é um site educativo desenvolvido como projeto escolar para o programa Agrinho 2026. O objetivo principal é ensinar como o Agro 4.0 está transformando o campo, conectando o mundo físico ao digital para produzir mais alimentos de forma inteligente e protegendo o meio ambiente.

O design do site foi criado com uma estética Futurista e Frutiger Aero, misturando a natureza (tons de verde) com a tecnologia (tons de azul brilhante, transparências e efeito de vidro).

---

## 🚀 Sobre o Projeto e Recursos do Site

O site é dividido em seções interativas e responsivas para facilitar o aprendizado:

* **Header (Cabeçalho):** Menu de navegação fixo com efeito de vidro (glassmorphism) para navegar facilmente entre as seções.
* **Hero Section (Início):** Um banner de destaque com título iluminado e uma introdução ao conceito do Agro 4.0.
* **Inovações no Campo:** Cards explicativos sobre Drones, Sensores IoT, Tratores com Piloto Automático e Inteligência Artificial.
* **Sustentabilidade:** Uma área dedicada a explicar como a tecnologia ajuda a economizar água, reduzir insumos químicos e diminuir a poluição.
* **Desafio Agro Digital (Quiz):** Um jogo interativo com perguntas de múltipla escolha para testar o conhecimento do usuário, com correção automática e pontuação na tela.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído usando tecnologias web fundamentais:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização visual avançada, variáveis de cores, layouts em Flexbox e responsividade para celulares e computadores.
* **JavaScript:** Lógica de programação para capturar as respostas do formulário, calcular a pontuação do Quiz e exibir o resultado dinamicamente.

---

## 📁 Estrutura de Arquivos

A organização do código foi feita da seguinte maneira:

* `index.html`: Contém toda a estrutura e os textos do site.
* `style.css`: Responsável por todo o design, cores, efeitos de brilho e layout.
* `script.js`: Arquivo com a lógica do Quiz interativo.
